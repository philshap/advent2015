# Solutions for [Advent of Code 2015](https://adventofcode.com/2015)

## Clojure solutions

Run using
```bash
$ clj -X main/-main
```

## Java Solutions

- compile:
```shell
find src -name *.java | xargs javac -d out
```

- run
```shell
SESSION_TOKEN="<AoC session token>" java -cp out advent2015.Main
```

- run tests
 ```shell
java -cp out advent2015.Main test
```
